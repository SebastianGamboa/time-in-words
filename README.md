# time-in-words

Local installation

You need Java 11 installed

Clone
--------

```sh
git clone https://github.com/SebastianGamboa/time-in-words.git
```

Project folder
--------

```sh
cd time-in-words
```

Run
--------

```sh
./mvnw spring-boot:run
```

Access
--------

```
http://localhost:8080
```

Try on the server
--------

```
https://time-in-words.herokuapp.com
```
